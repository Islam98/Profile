Used Android Studio, NodeJS and MongoDB to implement a library project. Where there are two phases for in the application. Users phase where they view available items of the library and select items to buy and place to obtain their selected items, and an Admin phase where an admin can add new items to list for users and update available quantity in stock (quantity selected by users automatically subtracted from the quantity in the database so the admin will update if new quantities arrived. Also, admin can view all users, update their purchases' state whether they paid or not yet. The project was a course project and implemented individually. Got a grade of A in that course.
